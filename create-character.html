キャラクター JSON 作成ツール

フォームを入力して characters.json に追加するデータを生成します

👤 基本情報 BASIC
URLに使用。英小文字のみ（例: furina）
?
未選択
📊 基礎ステータス STATS
⚔️ 天賦 TALENTS

通常攻撃・元素スキル・元素爆発を追加できます。
各天賦に倍率項目（Lv.1〜15）を無制限に追加できます。

🌟 固有天賦 PASSIVE
✨ 命ノ星座 C1 〜 C6
{ } JSON プレビュー 未生成
📖 使い方

① フォームを入力
基本情報・ステータス・天賦を入力します。
入力するたびに右のJSONが自動更新されます。

② JSONをコピー or ダウンロード
生成されたJSONを取得します。
ダウンロードすると {id}.json として保存されます。

③ characters.json に追加
data/characters.json の
"characters": [...] 配列内に
コピーしたJSONオブジェクトを追加します。

📁 画像を用意する場合 キャラクター画像は
images/characters/{id}.png
に配置してください。

元素アイコンは
images/elements/{element}.svg
に配置されています。
